# 安装

### 直接下载

查看 [dist 目录](https://github.com/vuejs/vue-router/tree/dev/dist).~~注意，dist 目录下的文件是最新稳定版，不会同步更新到 `dev` 分支上的最新代码~~

### CDN
暂时没有之后提供

### NPM

``` bash
npm install vue-router
```

可以使用 CommonJS 模块规范：

``` js
var NovaRouter = require('vue-router')
```

也可以使用ES6的模块规范引入

```
import NovaRouter from 'nova-router'
```
